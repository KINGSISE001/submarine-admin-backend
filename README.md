## 项目简介


致力于使用最流行最主流的前后端技术，对于各种使用场景都有相应的解决方案，对需求的变化充满信心。

## 技术选型

| ⚡ **后端**      | ⚡ **前端**        |
| ---------------- | ------------------ |
| springboot 2.3.0 | Vue                |
| mybatis          | Element UI         |
| mybatis-plus     | axios              |
| spring security  | vue-admin-template |
| mapstruct        | Webpack            |
| lombok           |                    |

## 系统功能

- 权限管理
- 机构管理
- 角色管理
- 用户管理
- 字典管理
- 存储管理
- 定时任务
- SQL监控
